# Python for Data Analysis

📖 **Course website: <https://pfaffelh.github.io/python_for_data/>** (HTML, downloadable Jupyter notebooks, and a [PDF](https://pfaffelh.github.io/python_for_data/python_for_data.pdf)).

These are the course notes for a course on Python for data analysis in summer 2026 at the University of Freiburg, Germany. No prior Python knowledge is assumed; the notes go from the basics (data types, control flow, libraries) to tools for data analysis (numpy, pandas, matplotlib, scipy.stats) and a small project.

## Building the notes

The notes use [`quarto`](https://quarto.org/docs/get-started/). After installing it, render a single chapter (to PDF and `.ipynb`) with

```
quarto render 02_simple_data_types.qmd
```

or build everything (HTML book, PDF, and notebooks) at once with

```
quarto render
```

This takes a few minutes. To browse the result locally, serve the output directory and open it in your browser:

```
python -m http.server --directory docs
```

The Python packages needed to execute the code cells are listed in [`requirements.txt`](requirements.txt); a virtual environment is recommended.

## How it is published

On every push to `master`, a [GitHub Action](.github/workflows/publish.yml) renders the book and publishes it to the `gh-pages` branch, which serves the course website above. The downloadable notebooks are generated by the `quarto` post-render hook [`render_notebooks.sh`](render_notebooks.sh).
